Context Window Size

The maximum number of tokens (words or word pieces) that a model can process at once. Larger context windows allow the model to understand and maintain coherence across longer texts.

Intelligence Score

A comprehensive measure of a model's cognitive capabilities, including reasoning, analysis, and problem-solving abilities. Higher scores indicate better performance on complex intellectual tasks.

Input Price

The cost per million tokens for processing input text. This is what you pay for the text you send to the model for analysis or completion. Lower input prices reduce costs for large-scale processing.

Output Price

The cost per million tokens for generated text. This is what you pay for the model's responses. Output prices are typically higher than input prices and significantly impact overall operational costs.

Output Speed

Measured in tokens per second, this indicates how quickly a model generates responses. Higher speeds result in faster response times and better real-time performance, crucial for interactive applications.

Reasoning Capabilities

A model's ability to analyze complex information, make logical connections, and solve problems. Higher reasoning capabilities indicate better performance on tasks requiring critical thinking and deduction.

Cost Efficiency

The overall economic value, considering both input and output costs relative to performance. More cost-efficient models deliver strong capabilities while maintaining reasonable operational expenses.

MMLU Score

A standardized benchmark measuring a model's performance across multiple fields including science, humanities, and professional knowledge. Higher scores indicate broader and deeper understanding.
